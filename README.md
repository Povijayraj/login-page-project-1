# Registration Form with Real-Time Validation

A simple and responsive Registration Form built using HTML, CSS, and JavaScript. The form includes fields for Name, Email, and Password, and performs real-time validation to enhance user experience and ensure accurate data entry.

## Features

* Real-time input validation using JavaScript
* Name field validation to prevent empty submissions
* Email format validation using regular expressions
* Password strength validation with minimum character requirements
* Instant error messages and user feedback
* Clean and responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How It Works

As users enter their information, JavaScript validates each field instantly. Error messages are displayed when invalid data is detected, and the form can only be submitted when all inputs meet the required criteria.

## Purpose

This project demonstrates fundamental form handling, DOM manipulation, and client-side validation techniques in JavaScript, making it a great beginner-friendly web development project.
